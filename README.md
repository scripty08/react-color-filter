# @scripty/react-skeleton

# Description

react-skeleton component.

# Usage
```bash
npm install -s @scripty/react-skeleton
```

##### Client: Example.jsx

```javascript
import React from 'react';
import { ReactSkeleton } from '@scripty/react-skeleton';

export const Example = () => {

    return (
        <ReactSkeleton />
    );
};
```
